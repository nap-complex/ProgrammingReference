How to Log On to Freenode

Using HexChat on Linux


1. Open HexChat

2. Click on HexChat drop down menu

3. Click on NetWork List

4. If Freenode is highlighted click <connect>
( else, highlight Freenode then click connect )

5. let it settle a bit before typing anything

6. on text line type: /msg NickServ identify <password>
( current password is "whatthefuck" )

7. To exit type: /quit
