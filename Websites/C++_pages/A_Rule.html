Forcing Numbers

NOTE: Because floating-point constants are normally stored in memory as doubles, some compilers issue a warning message when you assign a floating-point constant to a float variable. For example, if num is a float, the following statement might cause the compiler to generate a warning message:
num = 14.725;
You can suppress the error message by appending the f suffix to the floating-point constant, as show here:
num = 14.725f;
If you want to force a value to be stored as a long double, append an L to it, as shown here:
1034.56L
