Rename Multiple Files

Append Something To Start of Multiple Files:


One Line Version

$ for f in *.jpg; do mv "$f" "test_$f"; done


Multi Line Version

$ for f in *.jpg

> do

> mv "$f" "test_$f"

> done


This will turn "001.jpg" into "test_001.jpg"


Rename using 'Rename'

$ rename 's/replacee/replacer/g' *.jpg
